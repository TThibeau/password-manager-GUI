<!-- GETTING STARTED -->
## Getting Started
Install the prerequisite libraries: tkinter and pyperclip.
### Prerequisites

Installing tkinter.
  ```sh
  pip install tk
  ```

Installing pyperclip.
  ```sh
  pip install pyperclip  
  ```
